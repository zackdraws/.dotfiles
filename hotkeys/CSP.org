*** Animation Shortcuts
| Action               | Shortcut        |
| -------------------- | --------------- |
| New Animation Cel    | Shift + G       |
| Select Previous Cel  | A               |
| Select Next Cel      | D               |
| Insert Frame         | Shift + X       |
| Go to Previous Frame | C               |
| Go to Next Frame     | V               |
| New Animation Folder | Shift + Alt + A |
| Play                 | Shift + Q       |

| Zoom Out            | 1         |
| Decrease Brush Size | Shift + Z |
