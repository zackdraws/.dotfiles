#+title: TVPaint - Hotkeys
*** Drawing
| Action                  | Shortcut  |
| ----------------------- | --------- |
| Stroke                  | D         |
| Color                   | B         |
| Lasso                   | L         |
| Color Picker            | ''        |
| Size Change             | Z         |
| Flip/Switch             | U / I     |
| Flip Drawing Horizontal | [         |
| Flip Drawing Vertical   | ]         |
| Rotate 15               | {         |
| Rotate                  | 7 / 8     |
| Notes                   | !         |

*** Timeline and Playback
| Action                         | Shortcut  |
| ------------------------------ | --------- |
| Previous/Next Drawing          | , / .     |
| Previous Panel                 | 3         |
| Next Panel                     | 4         |
| Delete Panel                   | 5 / 6     |
| Output Full Play               | Shift + V |
| Hide Timeline                  | ``        |
| Up Layer                       | Q         |
| Down Layer                     | A         |
| Toggle Bookmark                | C         |
| New Window                     | M         |
